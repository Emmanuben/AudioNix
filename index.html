🎵 AudioNix – Spotify Playlist Companion

AudioNix is a desktop tool that helps users manage and archive their Spotify playlists with ease. Built using SpotDL, FFmpeg, and CustomTkinter, it integrates with your Spotify Developer account for a personal and flexible music experience.

🔧 Key Features

  • Import public playlists via Spotify URL
  • Modern, ad-free desktop interface
  • Personal use only – authenticated via your Spotify Developer credentials
  • Offline playlist archiving powered by open-source tools
